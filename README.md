# SNAKE

![Cover](https://github.com/ecgitt/snakereimagined/blob/main/SNAKE%20Screenshots/snake.png?raw=true)


## Controls:

WASD or arrow keys to move Snake.

Click screen to begin the game.

Enter to restart the game.

## How to play:

Download the compiled zip to run the .exe in Windows.
